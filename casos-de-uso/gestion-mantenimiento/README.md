# Sistema de Gestión de Solicitudes de Mantenimiento de Infraestructuras

## Caso Práctico

Una empresa de servicios desea implantar un sistema para gestionar las **solicitudes de mantenimiento de infraestructuras** (electricidad, climatización, fontanería, equipos industriales).

Los empleados pueden registrar solicitudes indicando el tipo de incidencia, ubicación, descripción detallada y nivel de prioridad.
El responsable de mantenimiento clasifica las solicitudes, asigna técnicos y planifica intervenciones.

Los técnicos actualizan el estado de las tareas, registran el tiempo empleado y describen la intervención realizada. El sistema debe permitir el seguimiento del estado de cada solicitud, generar historiales de mantenimiento por ubicación y emitir notificaciones automáticas ante cambios de estado o retrasos.
